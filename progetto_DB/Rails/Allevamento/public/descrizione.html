L'obiettivo del progetto è:

1 - la creazione di una base di dati che raccolga, organizzi e conservi le informazioni riguardanti un allevamento di bovini.

2 - la creazione di una interfaccia web attraverso la quale un operatore possa svolgere alcune operazioni con la base di dati.

Le operazioni richieste all'interfaccia web sono:

- l'inserimento delle informazioni riguardanti l'allevamento,

- la visualizzazione di liste di informazioni riguardanti l'allevamento secondo opportuni filtri,

- la visualizzazione dei dati completi di ogni singola mucca attraverso uno schema.

L'obiettivo verrà perseguito scrivendo una libreria per la manipolazione dei dati del database in Ruby, per poi usare Ruby on Rails esclusivamente per il collegamento ad una interfaccia web.
